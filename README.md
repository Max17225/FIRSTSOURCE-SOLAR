# FIRSTSOURCE SOLAR - Corporate Website

This repository contains the front-end source code for the FIRSTSOURCE SOLAR website, a corporate platform showcasing their renewable energy and EPCC solutions. 

## Project Overview
This project was developed by working closely with the client to translate their brand identity and business requirements into a responsive, dark-themed industrial web design. 

## Development Note
Please note that the primary focus during this project's development cycle was rapid iteration, visual design, and meeting client deliverables. As a result, strict file organization, code modularity, and repository structuring were not the main priorities. The codebase serves primarily as the delivered front-end product.

## Tech Stack
* HTML5
* CSS3
* Vanilla JavaScript
