# FIRSTSOURCE SOLAR - Corporate Website

This repository contains the front-end source code for the FIRSTSOURCE SOLAR website, a corporate platform showcasing their renewable energy and EPCC solutions.

## Project Overview
This project was developed in collaboration with the client to translate their brand identity and business requirements into a responsive, dark-themed industrial web design.

## Development Note
During the development phase, the website was hosted and managed using the Plesk hosting control panel. Changes were made locally and then uploaded directly to the Plesk File Manager for deployment and testing.

Because the hosting workflow relied on manual file uploads rather than Git-based deployment, version control was not integrated into the development process. As a result, the complete finalized codebase was pushed to this repository as a single commit after the project was completed.

This repository therefore represents the final delivered front-end implementation of the website.

## Tech Stack
- HTML5  
- CSS3  
- Vanilla JavaScript
